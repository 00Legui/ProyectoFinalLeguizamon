# 🛍️ Simulador de eCommerce

## 📝 Descripción
Un simulador de tienda en línea desarrollado con JavaScript vanilla que permite a los usuarios navegar productos, filtrarlos por categoría y precio, agregarlos al carrito y gestionar sus compras.

## ✨ Características
- Catálogo de productos con imágenes y detalles
- Filtrado de productos por categoría y precio
- Carrito de compras con persistencia local
- Modo oscuro/claro
- Diseño responsive
- Sistema de notificaciones
- Gestión de stock en tiempo real

## 🚀 Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
- SweetAlert2 para notificaciones
- LocalStorage para persistencia de datos

## 🛠️ Instalación
1. Clona el repositorio:

bash
git clone https://github.com/00Legui/ProyectoFinalLeguizamon.git

2. Abre el archivo `index.html` en tu navegador web

## 💻 Uso
- Navega por los productos disponibles
- Utiliza los filtros para encontrar productos específicos
- Agrega productos al carrito
- Gestiona el carrito (eliminar productos, vaciar carrito)
- Finaliza la compra
- Cambia entre modo oscuro y claro según preferencia

## 🔍 Funcionalidades Principales
- `loadProducts()`: Carga inicial de productos
- `renderProducts()`: Muestra productos en pantalla
- `addToCart()`: Agrega productos al carrito
- `applyFilters()`: Aplica filtros de categoría y precio
- `toggleTheme()`: Cambia entre modo oscuro y claro
